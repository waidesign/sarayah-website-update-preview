# Prompt Templates

Reusable checklists for recurring task types on this project. These are
reference material to read before starting the task type in question — not
literal scripts to paste verbatim, and not something that changes who's
doing the work (see `../identity.md`).

- [`code-generation.md`](code-generation.md) — building new components/pages
- [`code-review.md`](code-review.md) — reviewing changes before calling them done
- [`debugging.md`](debugging.md) — investigating a bug on this project specifically
- [`project-examples.md`](project-examples.md) — worked examples from this
  codebase's real history, for calibration
- [`seo.md`](seo.md) — the SEO trade-off from the SPA decision, and what to
  do about it page by page
